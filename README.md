## 中文

### 📋 项目简介

QuickCmd 是一个功能强大的跨平台命令执行工具，提供图形化界面来快速执行常用系统命令。无需记忆复杂的命令行参数，只需点击按钮即可完成各种系统管理任务。

### ✨ 主要特性

- 🌍 **跨平台支持** - 完美支持 Windows、Linux、macOS 三大操作系统
- ⚡ **预置命令** - 内置数十个常用系统命令，开箱即用
- 🔧 **自定义命令** - 支持添加、编辑、删除自定义命令
- 📝 **变量支持** - 命令支持变量参数，执行时动态输入
- 🎨 **主题切换** - 支持日间/夜间两种主题模式
- 📋 **实时输出** - 命令执行结果实时显示
- 💾 **配置持久化** - 自定义命令自动保存到本地
- 🎯 **分类管理** - 命令按功能分类，查找更便捷

### 🚀 快速开始

#### 环境要求

- Windows 7或更高版本

### 📖 使用指南

#### 1. 执行预置命令

1. 启动程序后，根据当前操作系统自动切换到对应标签页
2. 在分类中找到需要的命令
3. 点击按钮即可执行
4. 执行结果会在底部的输出区域实时显示

#### 2. 添加自定义命令

1. 切换到 "⚡ 自定义命令" 标签页
2. 点击 "➕ 添加命令" 按钮
3. 填写命令信息：
   - **命令名称**：给命令起一个易记的名字
   - **命令内容**：输入要执行的命令
   - **变量配置**（可选）：添加需要动态输入的变量

**示例：**
```
命令名称：测试网络连接
命令内容：ping -n {count} {host}
变量：
  - count (描述: 测试次数, 默认值: 4)
  - host (描述: 目标主机, 默认值: www.baidu.com)
```

#### 3. 使用变量

在命令中使用 `{变量名}` 格式定义变量，执行时会弹出对话框让你输入具体值。

**变量配置说明：**
- **变量名**：在命令中引用的名称
- **描述**：变量的说明文字
- **默认值**：预填充的默认值（可选）

#### 4. 管理自定义命令

- **编辑命令**：点击命令卡片上的 "✏️ 编辑" 按钮
- **删除命令**：点击命令卡片上的 "🗑️ 删除" 按钮
- **执行命令**：点击命令卡片上的 "▶️ 执行" 按钮

#### 5. 主题切换

点击右上角的主题切换按钮：
- 🌙 夜间模式 - 深色主题，适合夜间使用
- ☀️ 日间模式 - 浅色主题，适合白天使用

### 🔧 配置文件

自定义命令保存在 `custom_commands.json` 文件中，格式如下：

```json
[
  {
    "name": "测试网络",
    "command": "ping -n {count} {host}",
    "variables": [
      {
        "name": "count",
        "description": "测试次数",
        "default": "4"
      },
      {
        "name": "host",
        "description": "目标主机",
        "default": "www.baidu.com"
      }
    ]
  }
]
```

### 💡 使用技巧

1. **批量命令**：在 Windows 中使用 `&` 连接多个命令，在 Linux/macOS 中使用 `;` 或 `&&`
2. **输出重定向**：可以在命令中使用 `>` 或 `>>` 将输出保存到文件
3. **管理员权限**：某些命令可能需要管理员权限才能执行
4. **命令超时**：默认命令执行超时时间为 30 秒
5. **清空输出**：点击输出区域右上角的 "🗑️ 清空" 按钮清除历史输出

### 🛠️ 技术架构

- **界面框架**：PyQt6
- **多线程**：QThread 实现命令后台执行
- **进程管理**：subprocess 模块执行系统命令
- **数据存储**：JSON 格式存储配置
- **跨平台**：platform 模块识别操作系统

### 📝 开发说明

#### 项目结构

```
let_you_hand/
├── QuickCMD.py              # 主程序文件
├── main.py                  # 程序入口（与 QuickCMD.py 相同）
├── custom_commands.json     # 自定义命令配置文件
└── README.md               # 项目说明文档
```

#### 核心类说明

- **CommandExecutor**：命令执行线程类，负责后台执行命令并实时返回输出
- **AddCommandDialog**：添加/编辑命令对话框类
- **LetYouHandApp**：主窗口类，包含所有 UI 和业务逻辑

### 🐛 常见问题

**Q: 为什么某些命令执行失败？**
A: 可能是权限不足，尝试以管理员身份运行程序。

**Q: 中文输出显示乱码怎么办？**
A: Windows 系统已自动使用 GBK 编码处理，如仍有问题请检查系统编码设置。

**Q: 如何备份自定义命令？**
A: 直接复制 `custom_commands.json` 文件即可。

**Q: 命令执行超时怎么办？**
A: 当前超时时间为 30 秒，如需执行耗时较长的命令，建议在终端中手动执行。
